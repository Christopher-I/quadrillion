## Important Insights & Lessons

### 1. "Industrial" Name Can Be Misleading
Templates with "Industrial" in the name (Industrial X, IndustryFlow) are built for **manufacturing companies**, not PE firms that invest in them. Critical distinction.

**George wants**: "Institutional Industrial" AESTHETIC (calm, credible, precise)
**NOT**: Templates for industrial/manufacturing COMPANIES

### 2. Figma Files Are Always a Plus
User initially thought I positioned Figma files as negative for Foresight.

**Correction**: Figma files are ALWAYS a benefit, never a drawback. They provide:
- Design reference for customization
- Future flexibility for updates
- Understanding of design intent

### 3. Free ‚â† Better
CrestFund is free but scored 7.5/10 (below cutoff) because:
- Bold dark aesthetic doesn't match George's "calm, credible" vision
- Free = less unique (many others might use it)
- Quality matters more than price for this project

### 4. Demo Strategy Evolution
User initially worried about showing templates directly (buyer's remorse risk).

**Solution**: Position as expert curator + executor
- Value = curation + customization + execution (not just finding)
- Templates are "raw material" (20%), custom work is value (80%)
- Include URLs but emphasize the work ahead
